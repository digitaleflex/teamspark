# Workflows Utilisateurs - TeamSpark

## 1. Workflow d'inscription et de première connexion

### Étapes :
1. L'utilisateur accède à la page d'accueil
2. L'utilisateur clique sur "Commencer l'aventure"
3. L'utilisateur remplit le formulaire d'inscription (nom, email, mot de passe)
4. L'utilisateur reçoit un email de validation
5. L'utilisateur clique sur le lien de validation dans l'email
6. L'utilisateur est redirigé vers la page de connexion
7. L'utilisateur se connecte avec ses identifiants
8. L'utilisateur accède au tableau de bord

## 2. Workflow de connexion

### Étapes :
1. L'utilisateur accède à la page de connexion
2. L'utilisateur entre son email et mot de passe
3. L'utilisateur clique sur "Se connecter"
4. Si les identifiants sont corrects, l'utilisateur est redirigé vers le tableau de bord
5. Si les identifiants sont incorrects, un message d'erreur s'affiche

## 3. Workflow de réinitialisation de mot de passe

### Étapes :
1. L'utilisateur clique sur "Mot de passe oublié" sur la page de connexion
2. L'utilisateur entre son adresse email
3. L'utilisateur reçoit un email avec un lien de réinitialisation
4. L'utilisateur clique sur le lien et est redirigé vers la page de réinitialisation
5. L'utilisateur entre un nouveau mot de passe
6. L'utilisateur confirme le nouveau mot de passe
7. L'utilisateur est redirigé vers la page de connexion

## 4. Workflow de gestion du profil

### Étapes :
1. L'utilisateur clique sur son avatar dans le header
2. L'utilisateur sélectionne "Profil" dans le menu
3. L'utilisateur modifie ses informations (nom, email, avatar)
4. L'utilisateur clique sur "Enregistrer"
5. Les modifications sont sauvegardées et un message de confirmation s'affiche

## 5. Workflow de création d'équipe

### Étapes :
1. L'utilisateur accède au tableau de bord
2. L'utilisateur clique sur "Créer une équipe"
3. L'utilisateur remplit le formulaire (nom de l'équipe, description)
4. L'utilisateur clique sur "Créer"
5. L'équipe est créée et l'utilisateur est redirigé vers la page de l'équipe

## 6. Workflow d'invitation de membres

### Étapes :
1. L'utilisateur accède à la page d'une équipe
2. L'utilisateur clique sur "Inviter des membres"
3. L'utilisateur entre les adresses email des membres à inviter
4. L'utilisateur sélectionne les rôles pour chaque membre
5. L'utilisateur clique sur "Envoyer les invitations"
6. Les invitations sont envoyées par email

## 7. Workflow de consultation des analytics

### Étapes :
1. L'utilisateur accède au tableau de bord d'une équipe
2. L'utilisateur sélectionne l'onglet "Analytics"
3. L'utilisateur peut voir les graphiques de performance
4. L'utilisateur peut filtrer par période
5. L'utilisateur peut exporter les rapports

## 8. Workflow de réception d'insights IA

### Étapes :
1. L'utilisateur accède au tableau de bord d'une équipe
2. L'utilisateur voit les notifications d'insights dans la sidebar
3. L'utilisateur clique sur une notification
4. L'utilisateur voit les détails de l'insight
5. L'utilisateur peut suivre les recommandations
